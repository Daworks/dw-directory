이 플러그인은 워드프레스에서 디렉토리 서비스를 할 수 있도록 고안되었습니다.

== Description ==

This plugin is designed to enable directory services in WordPress.
It was developed for use in introducing the local restaurants in the local community site or for introducing the work place in the church homepage.

이 플러그인은 워드프레스에서 디렉토리 서비스를 할 수 있도록 고안되었습니다.
지역 커뮤니티 사이트에서 지역의 맛집 등을 소개하는 용도나 교회 홈페이지에서 교우 사업장을 소개하는 용도로 사용하려고 개발했는데 사용하기 나름이겠지요.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `dw-directory` folder to the `/wp-content/plugins/` directory (플러그인을 다운 받아 압축을 푼 후 `/wp-content/plugins/`에 업로드 하거나 플러그인 설치에서 직접 설치하세요.)
2. Activate the plugin through the 'Plugins' menu in WordPress.('Plugins' 메뉴에서 활성화 버튼을 클릭하여 플러그인을 활성화 하세요.)
3. Place shortcode <code>[dw-directory]</code> to page or post you want. (디렉토리 목록을 보여주기 위해서 숏코드 <code>[dw-directory]</code>를 페이지나 포스트에 붙여 넣은 후 저장하세요.)

== Frequently Asked Questions ==

= 플러그인 설치 후 어떻게 사용하나요? =

플러그인을 활성화 한 후 디렉토리를 표시할 페이지나 포스트에 숏코드 <code>[dw-directory]</code>를 입력한 후 저장하세요.

= 디렉토리 관리는 어떻게 하나요? =

대시보드로 로그인 한 후 관리자 메뉴에 '디렉토리 서비스'라는 메뉴에서 관리합니다.
홈페이지에 로그인 한 사용자가 자신의 사이트를 등록하면 관리자 메뉴 - 등록 대기 관리 페이지에 대기 상태의 아이템이 나옵니다. 등록 대기 리스트에서 등록 버튼을 클릭하면 등록됩니다.

= 디렉토리의 카테고리는 어떻게 분류하고 등록하나요? =

'디렉토리 서비스 - 카테고리 관리 메뉴'에서 카테고리를 추가, 수정, 및 삭제할 수 있습니다.

= 아이템 관리는 어떻게 하나요? =
'디렉토리 서비스 - 아이템 관리' 메뉴에서 개별적인 아이템에 카테고리를 할당하거나 새로 등록, 수정, 삭제 할 수 있습니다.
